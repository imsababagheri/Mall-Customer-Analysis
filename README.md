# Mall-Customer-Analysis
پروژه تحلیل داده‌های مشتریان فروشگاه

معرفی پروژه

در این پروژه داده‌های مربوط به مشتریان یک مرکز خرید بررسی شده‌اند. هدف اصلی، پیدا کردن رابطه بین ویژگی‌های مختلف مشتریان مانند سن، جنسیت و درآمد سالانه با میزان تمایل آن‌ها به خرید (Spending Score) است.

ابزارهای استفاده‌شده

- Python
- Jupyter Notebook
- Pandas

مراحل انجام کار

1. آماده‌سازی داده‌ها

در ابتدا ستون "CustomerID" حذف شد، زیرا فقط شناسه مشتریان بود و در تحلیل تأثیری نداشت.

2. تبدیل داده‌های متنی

برای انجام محاسبات، مقادیر ستون "Gender" به صورت عددی تبدیل شدند:

- Male = 0
- Female = 1

3. محاسبه همبستگی

برای بررسی ارتباط بین متغیرها، ماتریس همبستگی محاسبه شد و نتایج تا دو رقم اعشار گرد شدند تا خواندن آن‌ها ساده‌تر باشد.

نتایج و مشاهدات

ارتباط سن و امتیاز خرید

ضریب همبستگی بین سن و امتیاز خرید برابر با 0.33- بود. این نتیجه نشان می‌دهد که افراد جوان‌تر معمولاً امتیاز خرید بالاتری دارند و بیشتر خرید می‌کنند.

ارتباط درآمد سالانه و امتیاز خرید

ضریب همبستگی بین درآمد سالانه و امتیاز خرید حدود 0.01 به دست آمد. این مقدار بسیار نزدیک به صفر است و نشان می‌دهد رابطه مشخصی بین درآمد افراد و میزان خرید آن‌ها در این مرکز خرید وجود ندارد.

تأثیر جنسیت بر امتیاز خرید

ضریب همبستگی جنسیت و امتیاز خرید حدود 0.06 بود که مقدار کمی است. بنابراین جنسیت تأثیر قابل توجهی بر الگوی خرید مشتریان در این مجموعه ندارد.

جمع‌بندی

نتایج این تحلیل نشان می‌دهد که سن مشتریان نسبت به درآمد و جنسیت تأثیر بیشتری بر میزان خرید آن‌ها دارد. همچنین درآمد بالا لزوماً به معنای خرید بیشتر نیست و در این مجموعه مشتریان جوان‌تر فعالیت خرید بیشتری داشته‌اند.
